# TP3_Formulaire

Dans ce TP, l’objectif était de mettre en place un formulaire Symfony à partir d’un code HTML existant. Le projet a été initialisé avec Symfony 7 et Bootstrap 5.3 a été utilisé pour la mise en forme de l’interface. Une page produit a d’abord été créée en affichant les informations statiques du produit, puis le formulaire HTML a été transformé en un formulaire Symfony en utilisant un FormType dédié. Le formulaire permet de sélectionner la quantité et la couleur du produit et est géré côté serveur par un contrôleur Symfony. L’utilisation de Twig a permis de personnaliser l’affichage du formulaire tout en respectant l’architecture MVC de Symfony. Ce TP a permis de mieux comprendre le fonctionnement des formulaires, des contrôleurs et des vues dans un projet Symfony.
